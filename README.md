# 呼嚕呼嚕南京東路共生公寓 - 生活機能地圖

這是一個專為南京東路共生公寓設計的互動式生活機能網頁，以租屋處為中心，實時抓取並呈現周邊 500 公尺至 2 公里內的所有日常生活設施（便利商店、量販超市、連鎖藥妝、藥局、捷運站、公車站牌、加油站等），並提供直線距離與步行時間估算。

## 🌟 核心特色

1. **🚲 YouBike 2.0 即時車況串接**：直接串接臺北市政府 YouBike 2.0 即時 API，即時呈現周邊站點的**可借車數**與**可還車位數**（如：可借:15 / 可還:8）。
2. **🏪 智慧分店名稱與門牌對齊**：內建台北市主要品牌（全聯、家樂福、7-11、全家、萊爾富、屈臣氏、康是美、在地健保藥局等）的精確 Google Maps 門市對齊資料庫，保證顯示 100% 正確的官方門市資訊。
3. **📍 Google Maps 整合**：每筆地點皆提供一鍵直達 Google 地圖商家頁面（可即時查看星等、評論、營業時間）與步行導航路徑規劃。
4. **🗺️ 質感極簡視覺**：支援深色 (Dark) 與淺色 (Light) 主題切換，採用毛玻璃 (Glassmorphism) 質感卡片與響應式排版，手機與電腦皆可流暢使用。

## 📂 專案檔案結構

- `index.html` - 網頁主架構與控制面板
- `styles.css` - 現代化毛玻璃視覺設計與主題樣式
- `app.js` - 地圖核心邏輯、分類過濾器、即時資料處理與門市資料對齊

## ⚙️ 技術與資料來源

* **地圖框架**：[Leaflet.js](https://leafletjs.com/) (以 OpenStreetMap 為基礎圖磚)
* **圖標庫**：[Lucide Icons](https://lucide.dev/)
* **資料來源**：
  * 生活機能 POI：OpenStreetMap Overpass API
  * YouBike 2.0 資訊：臺北市政府交通局即時開放數據 API
  * 地址搜尋解析：Nominatim Geocoding API
